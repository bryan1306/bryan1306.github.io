# Análisis de Rendimiento Operativo

## Objetivo
Analizar datos operativos para identificar ineficiencias y oportunidades de mejora en procesos internos.

## Herramientas utilizadas
SQL | Python | Excel

## Proceso
- Limpieza y validación de datos
- Análisis de tiempos y volumen de operación
- Identificación de tendencias y desviaciones
- Construcción de métricas clave de desempeño

## Resultados
- Identificación de procesos con mayor variabilidad
- Detección de posibles cuellos de botella
- Recomendaciones basadas en datos para optimización
